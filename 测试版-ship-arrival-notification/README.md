## 新增功能

### 🚀 船只到港 Windows 系统通知

当您的船只抵达目的地时，会自动弹出 Windows 系统通知提醒您。

**功能特性**：
- ✅ 自动检测船只到达
- ✅ 显示船只注册号和目的地
- ✅ 10秒后自动关闭通知
- ✅ 点击通知可聚焦游戏窗口
- ✅ 防止重复通知

## 安装方法

### Chrome / Edge
1. 打开浏览器扩展页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. 开启"开发者模式"
3. 点击"加载解压的扩展程序"
4. 选择解压后的文件夹

### Firefox
1. 在地址栏输入 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件..."
3. 选择文件夹中的 `manifest.json` 文件

## 首次使用

1. 加载扩展后，浏览器会自动打开游戏页面
2. 当第一次有船只到达时，浏览器会弹出通知权限请求
3. 点击"允许"以启用系统通知功能

## 功能开关

在扩展设置中（输入 `XIT SET` 命令），找到"功能"标签页，可以找到：
- **NOTS：飞船抵达时发送 Windows 系统通知**

默认启用，可以随时关闭。

## 测试建议
