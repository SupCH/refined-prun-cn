Privacy Policy

Refined PrUn (Chinese Enhanced Edition)

Last Updated: January 6, 2026

Overview

Refined PrUn (Chinese Enhanced Edition) ("the Extension") is committed to protecting user privacy. This privacy policy explains how the Extension collects, uses, and protects your information.

Information Collection and Use

1. No Personal Information Collection

  • The Extension does NOT collect any personally identifiable information (PII)
  • The Extension does NOT track user behavior or usage patterns
  • The Extension does NOT transmit any data to external servers

2. Local Data Storage

The Extension uses Chrome's storage API to store the following data locally on your device only:

  • Feature toggle preferences (80+ configurable options)
  • Custom action package configurations (XIT ACT automation scripts)
  • UI layout preferences and language settings
  • Game-related data cached for performance optimization

3. Data Usage

All collected data is used exclusively to:

  • Persist user preferences across browser sessions
  • Enhance game interface functionality
  • Provide customized user experience within the game

4. Data Access and Security

  • All data is stored locally in your browser using Chrome's secure storage mechanism
  • No data is ever transmitted to third-party servers or external services
  • Only the Extension has access to the stored data
  • Users can clear all stored data at any time through browser settings

5. Third-Party Services

The Extension does NOT integrate with any third-party analytics, advertising, or tracking services.

6. Host Permissions

The Extension requires access to https://apex.prosperousuniverse.com/* only to:

  • Inject UI enhancements into the game interface
  • Intercept and process game API responses for data visualization
  • All operations are strictly limited to the official game domain

7. Network Requests

The Extension uses the webRequest permission to:

  • Monitor game API requests for real-time data updates
  • Enhance game data loading workflows
  • No external network requests are initiated by the Extension

Children's Privacy

This Extension is not directed to children under the age of 13. We do not knowingly collect information from children under 13.

User Rights

Users have the right to:

  • Access all data stored by the Extension through browser developer tools
  • Delete all Extension data by uninstalling the Extension or clearing browser storage
  • Opt-out of specific features through the Extension's settings panel

Compliance

This Extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) guidelines

Contact Information

For questions or concerns about this privacy policy, please contact:

  • GitHub: https://github.com/SupCH/refined-prun-cn/issues

概述

Refined PrUn 中文增强版（"本扩展"）致力于保护用户隐私。本隐私权政策说明了本扩展如何收集、使用和保护您的信息。

信息收集与使用

1. 不收集个人身份信息

  • 本扩展不会收集任何可识别个人身份的信息（PII）
  • 本扩展不会追踪用户行为或使用模式
  • 本扩展不会将任何数据传输至外部服务器

2. 本地数据存储

本扩展使用Chrome的 storage API 在您的设备本地存储以下数据：

  • 功能开关偏好设置（80+ 个可配置选项）
  • 自定义动作包配置（XIT ACT 自动化脚本）
  • UI 布局偏好和语言设置
  • 为性能优化而缓存的游戏相关数据

3. 数据用途

所有收集的数据仅用于：

  • 在浏览器会话之间保持用户偏好设置
  • 增强游戏界面功能
  • 在游戏内提供个性化用户体验

4. 数据访问与安全

  • 所有数据均使用Chrome的安全存储机制存储在您的浏览器本地
  • 数据绝不会传输至第三方服务器或外部服务
  • 只有本扩展能够访问存储的数据
  • 用户可随时通过浏览器设置清除所有存储的数据

5. 第三方服务

本扩展不集成任何第三方分析、广告或追踪服务。

6. 主机权限

本扩展仅需要访问 https://apex.prosperousuniverse.com/* 以便：

  • 向游戏界面注入 UI 增强功能
  • 拦截并处理游戏 API 响应以实现数据可视化
  • 所有操作严格限定在游戏官方域名内

7. 网络请求

本扩展使用 webRequest 权限来：

  • 监控游戏 API 请求以实现实时数据更新
  • 增强游戏数据加载工作流
  • 本扩展不会发起任何外部网络请求
